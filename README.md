# 🌱 GreenLens AI

แพลตฟอร์มวิเคราะห์พื้นที่ด้วย AI เพื่อแนะนำการฟื้นฟูสิ่งแวดล้อม

## ✨ คุณสมบัติหลัก
- 🤖 **AI Analysis**: วิเคราะห์พื้นที่ด้วย AI
- 🌳 **Plant Recommendation**: แนะนำพันธุ์ไม้ที่เหมาะสม
- 📊 **Impact Dashboard**: แสดงผลกระทบด้านสิ่งแวดล้อม
- 📈 **CO₂ Tracking**: ติดตามการดูดซับ CO₂
- 🎯 **SDG Alignment**: สนับสนุน UN SDG 15

Platform ของเรา https://greenlens-psi.vercel.app/
